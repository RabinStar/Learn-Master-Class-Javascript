Variable and Declaration Specifier

    Difference among const,let and var

  • Use var for function-scoped variables, though it is now less preferred.
  • Use let for block-scoped variables when you need to reassign them.
  • Use const for variables that should not be reassigned and remain constant after initializatio


Data Types

✅ Primitive Types → Number, String, Boolean, Null, Undefined, Symbol, BigInt 

      let a = 10;         // Number
let b = "Hello";    // String
let c = true;       // Boolean
let d = null;       // Null
let e;              // Undefined
let f = Symbol("x");// Symbol
let g = 10n;        // BigInt

✅ Non-Primitive Types → Object, Array, Function
